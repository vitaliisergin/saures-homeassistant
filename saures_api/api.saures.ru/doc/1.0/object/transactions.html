Платёжные транзакции

GET запрос

URL https://api.saures.ru/1.0/object/transactions

Параметры запроса
sid
uuid4 		идентификатор сессии
id
int 		id объекта
page
int 		номер страницы
step
int 		количество записей в ответе
Параметры ответа
transactions
[...] 		список транзакций
{id: id транзакци,
date: дата/время создания,
value: сумма,
status: статус транзакци,
status_display: статус для отображения пользователю,
text: дополнительная информация}
count
int 		всего транзакций
page
int 		номер страницы
step
int 		количество записей в ответе
Генератор HTTPie запроса (документация)

$ http GET https://api.saures.ru/1.0/object/transactions -v

Пример
$ http GET https://api.saures.ru/1.0/object/transactions sid=="92104027-908a-43dd-b18b-b06b918f696d" id=="10747" page=="1" step=="10" -v
